# 财务数据处理工具

自动化处理教育机构收入明细归类工具，根据sheet1的组织信息智能匹配sheet2中的明细记录。

## 功能特性

- 📊 读取Excel文件（支持.xlsx格式）
- 🔄 智能匹配算法：根据组织信息和金额自动归类
- ⚙️ 可配置的金额容差值（默认10元）
- 🖥️ 图形用户界面，操作简单直观
- 💾 保留原始文件，生成新的输出文件

## 安装步骤

1. 克隆或下载项目代码
2. 安装Python 3.8或更高版本
3. 安装依赖包：

```bash
pip install -r requirements.txt
```

## 使用方法

### 启动图形界面

```bash
python main.py
```

### 操作步骤

1. 点击"浏览..."按钮，选择要处理的Excel文件
   - 文件必须包含两个sheet：sheet1（组织信息）和sheet2（明细记录）
   - 支持的文件格式：.xlsx

2. （可选）调整金额容差值（默认10元）
   - 容差值用于判断金额是否匹配成功

3. 点击"开始处理"按钮
   - 程序将读取数据并进行智能匹配
   - 处理进度和日志将显示在界面下方

4. 查看处理结果
   - 处理完成后，会弹出提示框
   - 输出文件名为：output_原文件名.xlsx
   - 输出文件保存在原文件所在目录

## 输入文件格式

### Sheet1（组织信息）

| 组织 | 日期 | 当日金额 | 附件金 | 差异 |
|------|------|----------|--------|------|
| 组织A | 2024-01-01 | 1000.00 | 100.00 | 0.00 |
| 组织B | 2024-01-01 | 2000.00 | 200.00 | 0.00 |

### Sheet2（明细记录）

| 组织 | 日期 | 姓名 | 当日金额 |
|------|------|------|----------|
| 组织A | 2024-01-01 | 张三 | 500.00 |
|      | 2024-01-01 | 李四 | 500.00 |
| 组织B | 2024-01-01 | 王五 | 1000.00 |
|      | 2024-01-01 | 赵六 | 1000.00 |

**注意**：组织列为空表示该记录尚未归类

## 匹配算法说明

工具使用智能匹配算法，主要策略如下：

1. **日期分组**：首先按日期将记录分组
2. **组织匹配**：优先匹配同组织、同日期的记录
3. **金额核对**：检查有组织记录的总额是否与sheet1中的当日金额匹配
4. **无组织记录匹配**：如果金额不足，从同日期无组织记录中选择合适的记录填充
5. **冲突避免**：确保同一人名只出现在一个组织中

## 输出文件

输出文件包含两个sheet：
- Sheet1：保持不变（原始的组织信息）
- Sheet2：明细记录，组织字段已根据匹配结果更新

## 常见问题

### Q: 支持哪些Excel格式？
A: 仅支持.xlsx格式，不支持旧的.xls格式

### Q: 文件大小有限制吗？
A: 建议单个文件不超过10万行，内存限制取决于系统配置

### Q: 容差值设置多少合适？
A: 默认10元适合大多数场景，可根据实际数据精度调整

### Q: 处理后的数据准确吗？
A: 工具使用智能算法匹配，但建议人工抽查验证结果
